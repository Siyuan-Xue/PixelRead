PixelRead v1.1.1 refines the manual update status behavior beside the product identity.

Highlights:
- Keeps the small update icon beside `PIXELREAD` / `CODEX & XUE` on the home screen.
- Clears `LATEST` after 3 seconds and returns to the icon-only state.
- Clears available-update status after 3 seconds and returns to the icon-only state.
- Keeps the PDF/EPUB reader, recent-book resume, reader drawer, text sizing, and theme behavior unchanged.

Install note: if you still have the old `com.codexue.pixelread` prototype build, uninstall it before installing this private release.
